# Java Operadores Basicos

## Operadores Aritmeticos

| Operador           | Ejemplo |
|--------------------|---------|
| + (Suma)           | A + B   |
| - (Resta)          | A -B    |
| * (Multiplicacion) | A * B   |
| / (Division)       | A / B   |
| % (Modulo)         | A % B   |
| ++ (incremento)    | A++     |
| -- (decremento)    | B--     |

## Operadores Relacionales

| Operador               | Ejemplo |
|------------------------|---------|
| == (igual a)           | A == B  |
| != (no igual a)        | A != B  |
| > (mayor que)          | A > B   |
| < (menor que)          | A < B   |
| >= (mayor o igual que) | A >= B  |
| <= (menor o igual que) | A <= B  |
