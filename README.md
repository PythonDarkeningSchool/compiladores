# Java Operadores Basicos

## Operadores Aritmeticos

| Operador           | Ejemplo |
|--------------------|---------|
| + (Suma)           | A + B   |
| - (Resta)          | A -B    |
| * (Multiplicacion) | A * B   |
| / (Division)       | A / B   |
| % (Modulo)         | A % B   |
| ++ (incremento)    | A++     |
| -- (decremento)    | B--     |

## Operadores Relacionales

| Operador               | Ejemplo |
|------------------------|---------|
| == (igual a)           | A == B  |
| != (no igual a)        | A != B  |
| > (mayor que)          | A > B   |
| < (menor que)          | A < B   |
| >= (mayor o igual que) | A >= B  |
| <= (menor o igual que) | A <= B  |

## Operadores de Bit

| Operador                                          | Ejemplo |
|---------------------------------------------------|---------|
| & (bit and)                                       | A & B   |
| \| (bit or)                                       | A \| B  |
| ^ (bit XOR)                                       | A ^ B   |
| ~ (bit compliment)                                | ~A      |
| << (desplazamiento a la izquierda)                | A << 2  |
| >> (desplazamiento a la derecha)                  | A >> 2  |
| >>> (desplazamiento a la derecha de llenado cero) | A >>>2  |

## Operadoeres Logicos

| \|\| (logics or) | A \|\| B  |
